# 图标资源说明

本目录需要存放以下图标文件用于TabBar显示：

## 所需图标列表

| 文件名 | 用途 | 尺寸要求 |
|--------|------|----------|
| home.png | 首页图标（未选中） | 81x81px |
| home-active.png | 首页图标（选中） | 81x81px |
| goods.png | 货物图标（未选中） | 81x81px |
| goods-active.png | 货物图标（选中） | 81x81px |
| scan.png | 扫码图标（未选中） | 81x81px |
| scan-active.png | 扫码图标（选中） | 81x81px |
| record.png | 记录图标（未选中） | 81x81px |
| record-active.png | 记录图标（选中） | 81x81px |

## 获取图标的方式

### 方式一：使用图标生成器（推荐）
1. 在小程序中访问 `/pages/icon-generator/generator` 页面
2. 点击"生成图标"按钮
3. 图标将保存到本地存储
4. 在微信开发者工具中手动复制到本目录

### 方式二：使用在线图标库
1. 访问 [阿里巴巴矢量图标库](https://www.iconfont.cn/)
2. 搜索所需图标（如：首页、包裹、扫码、时钟）
3. 下载PNG格式，尺寸81x81px
4. 选中状态使用蓝色(#1890ff)，未选中使用灰色(#999999)

### 方式三：使用Material Icons
1. 访问 [Material Icons](https://fonts.google.com/icons)
2. 搜索对应图标
3. 导出为PNG格式

## 图标规格
- 格式：PNG
- 尺寸：81x81px（推荐）
- 未选中颜色：#999999
- 选中颜色：#1890ff
- 背景：透明
